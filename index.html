Aprenda Git e GitHub

Domine o versionamento de código e a colaboração em projetos

O que é Git?

Git é um sistema de controle de versão distribuído que permite rastrear mudanças no código e colaborar com outras pessoas.

O que é GitHub?

O GitHub é uma plataforma que hospeda repositórios Git na nuvem e facilita a colaboração entre desenvolvedores.

Principais Comandos

  • git init – Inicia um repositório Git.
  • git add . – Adiciona arquivos para serem versionados.
  • git commit -m "mensagem" – Salva as alterações.
  • git push – Envia as alterações para o GitHub.
  • git pull – Atualiza o repositório local com mudanças do GitHub.
